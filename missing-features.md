# Argon Modern 待修复与开发任务清单

本文档列出了 `argon-modern` 相比原版 `argon` 主题仍缺失、有问题或待优化的功能，用于追踪重构进度。

## 1. 核心功能差异 (Pending)

### 1.1 自动更新与统计
- **已修复**: `theme-update-checker` 及其相关的多源更新检测逻辑。
- **已修复**: 初次安装时的匿名安装量统计功能。

### 1.2 小工具与侧栏 (Widgets)
- **已修复**: `leftbar-author-links` 和 `leftbar-friend-links` 的 Walker 类已完全 OOP 化。
- **已修复**: `rightbar-tools` (右侧栏小工具) 在三栏布局下的前端布局兼容性已验证。
- **已修复**: 404 页面返回首页时的双页脚 (Double Footer) 冲突问题。

### 1.3 文章系统增强 (OOP 迁移)
- **已修复**: 字数统计与预计阅读时间逻辑已迁移至 `Template.php` 并集成至 Meta 显示。
- **已修复**: 目录序号开关 (`argon_show_headindex_number`) 的 CSS 渲染逻辑已集成至 `sidebar.php`。
- **已修复**: 文章目录 (Catalog) 的 `#post_content` 容器识别与 Pjax 自动重绘逻辑。
- **已修复**: 文章 Meta 信息在详情页与预览页的渲染逻辑统一化 (`render_article_meta`)。
- **已修复**: 文章详情页底部的“上下篇文章”导航卡片 UI 还原。
- **已修复**: 推荐文章 (Related Posts) 卡片的现代布局与缩略图支持。
- **已修复**: 文末附加内容 (Additional Content) 的占位符替换 (%url%, %link%, %title%, %author%) 与格式化支持。
- **已修复**: 文章标签 (Tags) 的深色 Badge 样式适配。
